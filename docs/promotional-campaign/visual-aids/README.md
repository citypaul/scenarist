# Visual Aids for Video Presentations

This directory contains diagrams and visual aids for the Scenarist promotional videos.

## Format

Diagrams are written in Mermaid markdown format. To use them:

1. **For slides**: Render using [Mermaid Live Editor](https://mermaid.live/) and export as PNG/SVG
2. **For blog posts**: Embed directly (GitHub/most markdown renderers support Mermaid)
3. **For video**: Screenshot from Mermaid Live or use OBS scene

## Rendering Tips

- Use a light background for video (better visibility)
- Export at 2x resolution for crisp display
- Keep diagrams visible for 5-10 seconds minimum
