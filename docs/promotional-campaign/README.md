# Scenarist Promotional Campaign

This folder contains all materials for the Scenarist promotional campaign.

## Structure

```
promotional-campaign/
├── PLAN.md              # Master campaign plan with all details
├── PROGRESS.md          # Progress tracking and status updates
├── README.md            # This file
├── blog-posts/          # Markdown blog post drafts
│   └── [numbered by video or standalone]
└── video-scripts/       # Detailed video scripts
    └── [numbered by video]
```
